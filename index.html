Data Processing Workflow

Automating data extraction, transformation, and publication.

Project Overview

This project demonstrates a robust data processing pipeline using Python and Pandas,integrated with GitHub Actions for continuous integration and deployment. It takes an Excelfile (data.xlsx),processes it through a Python script (execute.py),and publishes the processed results as a JSON file (result.json)to GitHub Pages.

The Python script ensures data quality and performs aggregation, while the CI/CD pipelineautomates linting, execution, and deployment, ensuring code quality and up-to-date results.

Key Components

  • Data Source: data.xlsx - An Excel spreadsheet containing raw input data.
  • Processing Logic: execute.py - A Python script (using Pandas) to read, clean, and transform the data.
  • Intermediate Data: data.csv - The converted CSV version of the input Excel file.
  • Output: result.json - The final processed data in JSON format, published to GitHub Pages.
  • Automation: .github/workflows/ci.yml - GitHub Actions workflow for linting, testing, and deployment.
